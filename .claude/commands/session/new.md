# New Session Command

새로운 작업 세션을 시작합니다.

## 사용법

```bash
/session:new                   # 대화형으로 시작
/session:new "로그인 기능 구현"    # 태스크와 함께 시작
```

## 인자

$ARGUMENTS

## 동작

1. **기존 세션 확인**
   - `.claude/session-state.json` 존재 시 경고
   - 저장하지 않은 변경 확인

2. **새 세션 초기화**
   - session-state.json 초기화
   - task-manager 에이전트 호출

3. **옵션 제시**
   - 인자가 있으면 OpenSpec 생성 시작
   - 인자가 없으면 옵션 표시:
     - 태스크 설명 입력
     - ROADMAP에서 선택
     - 기존 OpenSpec 계속

## 출력

```
═══════════════════════════════════════════════════════════════
NEW SESSION
═══════════════════════════════════════════════════════════════

옵션:
1. 작업할 내용을 설명해주세요
2. "status"로 ROADMAP 확인
3. "계속"으로 기존 태스크 이어서

═══════════════════════════════════════════════════════════════
```

## 주의사항

- 기존 세션이 있으면 `/session:save` 먼저 권장
- `/session:load`으로 기존 세션 복원 가능

## 관련 명령어

- `/session:load` - 기존 세션 복원
- `/workflow` - 워크플로우 실행
- `/session:save` - 세션 저장
